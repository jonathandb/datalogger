
Modbus Datalogger
=================
Welcome to the documentation for modbus datalogger.

The source code can be found on `github <http://github.com/jonathandb/datalogger/>`_ .

The functions are:
 * Behave as a modbus master and read data from slaves at configurable time intervals
 * Send data to the server
 * Configure datalogger
 * Extend software
