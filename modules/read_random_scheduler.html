read_random_scheduler module¶

class read_random_scheduler.ReadSensorScheduler(scheduler, packet_manager)[source][doc]¶

Bases: builtins.object

This class is used to debug the datalogger without the need of a modbus network. Instead of using ModbusSlaveReader it returns random data when it is expected to read data.
