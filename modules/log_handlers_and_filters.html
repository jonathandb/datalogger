log_handlers_and_filters module¶

class log_handlers_and_filters.LogSendStoreHandler(log_location)[source][doc]¶

Bases: logging.Handler

This handler decides what happens with the created logs messages. The logs can be sent to the server or/and stored locally. The sending and storing can done in configurable time intervals to limit wear of network connection and storage.

keep_logfile_in_max_limits()[source][doc]¶

Organises the logfiles. If a logfile is to big, it is stored under another name and a new logfile is created. If there are too many logfiles, the oldest one is removed. The total max log size is set in the configuration. The individual log file size is set in global variable SINGLE_LOG_FILE_SIZE .
